## About Cross Purpose

Cross Purpose is a transformative program that partners with adults experiencing poverty to help them discover and pursue their purpose in life. Our client portal provides a comprehensive platform for participants, coaches, and administrators to collaborate throughout the journey.

## Technical Stack

Built with modern, scalable technologies:

- **Frontend**: Next.js 14 with React 18
- **Backend**: API Routes with TypeScript
- **Database**: PostgreSQL with Prisma ORM
- **Hosting**: Vercel with Neon database
- **Integrations**: Salesforce, Track1099, Resend
